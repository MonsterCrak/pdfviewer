# 📚 PDF Viewer Web

Aplicación web para visualizar archivos PDF desarrollada en JavaScript.

## 🛠️ Tecnologías
- JavaScript
- PDF.js
- HTML/CSS

## ✨ Funcionalidades
- Visualización de documentos PDF
- Navegación entre páginas
- Zoom in/out
- Modo pantalla completa

## 🚀 Uso
```bash
# Abrir en navegador
# Solo abre el archivo index.html
```

## 📄 Requisitos
- Navegador moderno con soporte ES6
